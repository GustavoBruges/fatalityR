fatalityR

This package provides utility functions to view and plot data from the US National Highway Traffic Safety Administration’s Fatality Analysis Reporting System. For more details visit: https://www.nhtsa.gov/research-data/fatality-analysis-reporting-system-fars - FARS is a nationwide census providing NHTSA, Congress and the American public yearly data regarding fatal injuries suffered in motor vehicle traffic crashes.

This vignette describes the use of the functions contained in the packages. For this vignette you need to load the following packages:

Reading Data

This package includes 2013 raw data in the “inst/extdata” folder, you can access to it with the (system.file) function:

system.file("externaldata", "accident_2013.csv.bz2", package = "fatalityR")
## [1] "/home/daniel/Escritorio/fatalityAnalysis/inst/externaldata/accident_2013.csv.bz2"

The FARS data is a annual compilation in CSV format. This packages provides the function fars_read for reading data and return tbl data frame. You can use the (fars_read) function to import a dataset, you only need to supply the filename parameter. See the ((fars_read)) function for more details. An example of the functionality:

temp <- system.file("externaldata", "accident_2015.csv.bz2", package = "fatalityR")
dataset <- fars_read(temp)

Summarizing data

If you want to summarize how much observations by year-month occurred with the function, you only have to supply a integer vector with the years you want to analyze and it will return a wide format tibble with months on rows, years on columns and number of observations as values.

setwd(system.file("externaldata", package = "fatalityR"))
fars_summarize_years(2013:2015)

Plot data

You can plot the map of the count of accidents in a specific state in a specific year using fars_map_state.

setwd(system.file("externaldata", package = "fatalityR"))
fars_map_state(1,2015)
